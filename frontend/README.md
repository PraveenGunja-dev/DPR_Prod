# Adani Flow - Frontend

React + TypeScript frontend application built with Vite.

## Prerequisites

- Node.js 18+
- npm or yarn

## Installation

```bash
npm install
```

## Environment Setup

Create a `.env` file with:

```env
VITE_API_URL=http://localhost:3000
```

## Development

```bash
npm run dev
```

Opens at http://localhost:5173

## Production Build

```bash
npm run build
npm run preview
```

## Technology Stack

- **React 18** - UI Framework
- **TypeScript** - Type Safety
- **Vite** - Build Tool
- **TailwindCSS** - Styling
- **Radix UI** - Components
- **React Query** - Data Fetching
- **Handsontable** - Excel-like Tables
